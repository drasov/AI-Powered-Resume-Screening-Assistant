## AI-Powered Resume Screening Assistant

### Description
This project is an AI-Driven resume screening tool that analyzes resumes using NLP and semantic similarity to match candidates to job descriptions. It anonymizes resumes, embeds text using transformer models, and ranks candidates by relevance.

### Requirements
* Python 3.10+
* Install [`dependencies`](https://github.com/drasov/AI-Powered-Resume-Screening-Assistant/blob/main/requirements.txt):
`pip install -r server/requirements.txt`

### Running the Backend
From the project root:
`uvicorn server.main:app --reload `
